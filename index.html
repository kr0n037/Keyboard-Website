Installing the Arduino IDE and It's Required Libraries

The Arduino IDE is a simple IDE that we will use to compile and upload code to your 15-Key Macro Keyboard. The Arduino IDE is a beginner-friendly, free coding environmetn that has been developed and supported by the Arduino Company. This 15-Key Macro Keyboard draws inspiration from the Arduino Leonardo. This keyboard uses an Arduino Bootloader, and multiple Arduino Libraries. If you dont know what that means, dont worry about that for now! First, navigate to the Arduino IDE download page and download the appropriate software for the type of computer you are using.

Now that you have the Arduino IDE installed, we will need to install some libraries so that the code generated from the Code Generator will work without any errors. A library is something we install that makes writing code for certain applications easier. That way if you want to edit the generated code for this keyboard, you can! This board uses the Arduino Leonardo bootloader. A table with the microcontroller connections will be included at the end of this guide. This keyboard is 100% customizable.

Required Libraries

  1. Arduino - By Arduino Corp
  2. Keyboard - By Arduino
  3. Adafruit_NeoPixel - By AdaFruit
  4. Keypad - By Mark Stanely

All of these libraries are free to install. The Arduino and Keyboard libraries come pre-installed in the Arduino IDE! HERE is a tutorial on how to install libraries into the Arduino IDE. Ensure that all the required libraries have been installed prior to attempting to program your 15-Key Macro Keyboard!

Basics of Using the 15-Key Macro Keyboard

This section of the User's Manual reviews the basics of using your 15-Key Macro keyboard. This Macro Keyboard has 5 profiles and four animation modes (Classic, Solid Color, Dynamic Keys, 404 Animation Not Found) each animation mode contains multiple animations, except 404 Animation not found. Each button is customizable to the profile, Modifier Keys are also supported on this macro keyboard through the Code Generator.

Changing Profiles

  1. To change profiles you need to enter Configuration Mode. To enter 'Configuration Mode' press key 15.
  2. when you are in 'Configuration Mode' (all LED lights under the keys will be flashing GREEN) press buttons 1-5 to choose which profile you would like to enter.
    1. button 1 = Profile 1
    2. button 2 = Profile 2
    3. button 3 = Profile 3
    4. button 4 = Profile 4
    5. button 5 = Profile 5
  3. the Profile you are in will be represented by the LED lights on the left side of the keyboard. The lights represent the profile the keyboard is currently in.
